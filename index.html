Essential Commands

/seedcracker gui

Open config GUI (1.19.3+). Toggle finders, render mode, database & server mc-version.

/seedcracker data bits

Display current normal bits and liftable bits. Cracking starts at 32/40 bits respectively.

/seedcracker data clear

Clear all collected data without relogging — perfect for multi-world servers.

/seedcracker finder reload

Rescans already loaded chunks to detect structures that were missed.

/seed data restore — Restore previously saved structure data from config folder.

Installation Guide

1. Install Fabric Loader

Download the Fabric installer from fabricmc.net. Run it and select your Minecraft version (e.g., 1.21.1). Click "Install".

Screenshot placeholder: Fabric installer window with version selected.

2. Download SeedCrackerX

Grab the latest .jar file from the Downloads section below or from GitHub Releases. Choose the version matching your Minecraft.

Screenshot placeholder: Release page showing SeedCrackerX-2.15.6.jar.

3. Place the JAR in mods folder

Open your Minecraft directory (%appdata%/.minecraft/ on Windows, ~/Library/Application Support/minecraft/ on macOS, or your launcher's instance folder). If the mods folder doesn't exist, create it. Move the downloaded .jar file into mods.

Screenshot placeholder: mods folder with SeedCrackerX.jar and other mods.

4. (Optional) Install ViaFabric or Multiconnect

To connect to servers on lower Minecraft versions, also place ViaFabric or Multiconnect in the mods folder.

Screenshot placeholder: ViaFabric in mods folder.

5. Launch Minecraft with Fabric profile

In your launcher, select the Fabric profile and start the game. SeedCrackerX will load automatically. Use /seedcracker gui to configure.

Screenshot placeholder: In-game config GUI opened.
Frequently Asked Questions
❓ Mod not showing up / command not found

Make sure you're using the correct Fabric loader version for your Minecraft. Ensure the mod JAR is in the correct mods folder and not corrupted. Try reinstalling Fabric and the mod. If using a launcher like MultiMC or Prism, verify the instance has Fabric installed.

❓ Cracking doesn't start / bits stuck

Use /seedcracker data bits to check your progress. You need 40 liftable bits and 32 regular bits. If bits aren't increasing, try /seedcracker finder reload to rescan chunks. Ensure structures are correctly outlined (green box). Some structures like shipwrecks may not register if partially generated; explore more.

❓ Outdated mod version for my Minecraft

Check the Downloads table for version compatibility. If you're on a newer Minecraft version that isn't listed, you can use ViaFabric or Multiconnect to connect to older servers while running a compatible SeedCrackerX version on a newer client.

❓ Crash on launch / conflicts with other mods

Try removing other mods temporarily to isolate the issue. Ensure all mods are for the same Minecraft version and Fabric loader. Check the latest.log for error messages. Report issues on GitHub Issues with the log attached.

❓ How do I use the database feature?

Enable the database option in the config GUI (/seedcracker gui). The mod will automatically upload cracked seeds from servers with 10+ players to a public Google Sheet. You need a valid Minecraft license. The sheet is maintained by the community and helps map seeds across versions.

❓ Can I use SeedCrackerX on Spigot/Paper servers?

SeedCrackerX is a client-side Fabric mod. It works on any server (vanilla, Spigot, Paper, etc.) as long as you have the mod installed on your client. However, some server-side anti-cheat plugins may flag the mod; use at your own discretion.

❓ What are "liftable bits" and "regular bits"?

Liftable bits come from structures that can be "lifted" (like shipwrecks, pyramids, etc.) and are used for the structure-based cracking method. Regular bits come from end cities and other sources. You need 40 liftable bits and 32 regular bits to start the cracking process.

❓ How do I contribute to the seed database?

Simply enable the database option in config. When you crack a seed on a server with 10+ players, it will be automatically uploaded (anonymized). You can also manually submit seeds via the community Google Sheet link in the mod's config.
